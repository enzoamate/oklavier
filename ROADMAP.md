# Roadmap

The roadmap lives on GitHub Projects:

**👉 [github.com/users/enzoamate/projects/1](https://github.com/users/enzoamate/projects/1)**

Items are tracked as [issues labeled `roadmap`](https://github.com/enzoamate/oklavier/issues?q=is%3Aissue+label%3Aroadmap) with these fields:

- **Status** — Todo / In Progress / Done
- **Priority** — P0 critical / P1 high / P2 nice-to-have
- **Area** — Core API, Frontend, Agent, Helm, Docs, Infra
- **Target version** — v1.1, v1.2, v2.0, Backlog
- **Start date** / **Target date** — for the timeline view

The Project has three views:

- **Roadmap** — timeline / Gantt across versions
- **Board** — Kanban grouped by Status
- **Table** — full grid

## Influence the roadmap

- 👍 React on an existing [issue](https://github.com/enzoamate/oklavier/issues?q=is%3Aissue+label%3Aroadmap) — most-upvoted items get prioritized
- 💬 [Open a discussion](https://github.com/enzoamate/oklavier/discussions) before filing anything large
- ✋ Comment "I'd like to take this" on any issue to claim it
- 🔒 Security issues — see [SECURITY.md](SECURITY.md), do **not** open a public issue

## Versioning

[Semantic Versioning](https://semver.org/). Conventional Commits drive automatic version bumps via release-please. See [CONTRIBUTING.md](CONTRIBUTING.md).
